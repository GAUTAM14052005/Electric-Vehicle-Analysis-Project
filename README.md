# Electric-Vehicle-Analysis-Project
Python-based end-to-end EV Data Analysis using Pandas, Matplotlib, Seaborn, and Scikit-learn. Explores trends in electric vehicle adoption, range, pricing, and regional distributions. This Project demonstrates practical data science skills including data transforming, statistical analysis, and model evaluation. 

# Project Overview
This Project presents a comprehensive data analysis of electric vehicle (EV) data using Python. The goal is to uncover patterns, and build predictive models that provide insights into the the growing EV market.
# Objectives
1. Data Collection & Loading. 
2. Data Cleaning & Preprocessing.
3. Exploratory Data Analysis (EDA).
4. Feature Engineering.
5. Model Evaluation.
6. Insights Generation.
# Visualizations
Distribution of Electric Range (Histogram), Top 5 EV Models (Bar chart), Numeric features (Correlation Heatmap), Actual vs Predicted Values (Scatter plot). 
# Machine Learning Models Used: 
Linear Regression and Random Forest Regressor. 
# Key Insights:
1. BEVs outnumber PHEVs.
2. EV adoption is growing.
3. Electric Range varies widely.
# Dataset Source: 
The dataset used in this project is the Electric Vehicle Population Data. 
Source: data.gov
Data Format: CSV File (Comma-Separated-Values).
# Technologies Used:
Programming Language: Python
Data Manipulation: Pandas, NumPy
Data Visualization: Matplotlib, Seaborn
ML Model Evaluation: Scikit-learn.
Environment: Python IDLE / Jupyter Notebook  
# Conclusion & Future Scope
The project successfully demonstrated a complete data science pipeline applied to real-world electric vehicle data. The visulizations effectively highlighted key patterns such as the dominance of Tesla in the EV market, and the strong correlation between vehicle price and electric range.
Finally, a deeper analysis of battery technology trends over the years could reveal how advancements in battery capacity have shaped vehicle pricing and range improvements, providing valuable insights!
